The oldest repository here containing random code snippets.

## Macros
Contains a VB Excel Macro that can be used to convert Jira ticket (in the format XXX-1234) into hyperlinks (in the format xxx.atlassian.net/browse/XXX-1234)

## Snippets

**date countdown timer** - is a jQuery code that will read a timestamp value in the specified html element and create a countdown timer to that time if it is in the future or show that it has expired. This was used for a promotion expiry counter

**flower braces css** - is a css code that creates a neat flower brace over a block section to indicate a text annotation.

## hosted wifi

Contains a windows shell script that will create a hotspot wireless network with the indicted name and password.
